## 选项A：城市独特性深度挖掘（标准模板）

### 适用场景

- ✅ 首次访问的知名城市
- ✅ 需要全面介绍的旅游目的地
- ✅ 历史、文化、人物相对均衡的主题

### 时间分配

```
三、核心内容（8-10分钟）
├── 3.1 历史背景 (2-3分钟)
├── 3.2 文化特色 (3-4分钟)
└── 3.3 人物故事 (2-3分钟)
```

### 使用说明

直接使用主模板的第三部分，无需任何调整。这是最通用的模板。

### 示例笔记

- `Travel/格鲁吉亚_第比利斯_第1期.md` - 综合性城市介绍

## 选项B：小众秘境深度探索（探索型）

### 适用场景

- 🗺️ 真正小众、信息稀缺的目的地
- 🗺️ 需要探索过程的内容（徒步、探险、考古）
- 🗺️ "寻找"、"发现"性质的主题
- 🗺️ 边境、禁区、特殊地理区域

### 时间重新分配

```
一、开场钩子 (2-3分钟) - 强调"未知秘境"
二、抵达与初印象 (2-3分钟)
    ├── 1. 抵达过程（探索感）
    ├── 2. 城市氛围（神秘感）
    └── 3. 探索动机 ⭐ 新增
三、历史背景（精简版） (1-2分钟) ⬇️ 压缩
四、小众秘境深度探索 (10-12分钟)
    ├── 4.1 秘境一：[地点] (3-4分钟)
    │   ├── 探索过程
    │   ├── 秘境展示
    │   └── 发现与思考
    ├── 4.2 秘境二：[地点] (3-4分钟)
    │   └── （同上结构）
    └── 4.3 秘境三：[地点] (3-4分钟)
        └── （同上结构）
五、隐藏亮点深度推荐 (4-5分钟)
六、探索心得与个人思考 (2-3分钟)
七、实用信息（探索版）(1-2分钟)
八、结尾与探索总结 (1分钟)
```

- **人物群像**: 结尾回顾时要有情感高潮

## 混合使用指南

### 什么时候可以混合模块？

✅ **可以混合**的场景：

- 标准模板的历史部分（3.1）可以替换为历史型的深度模块
- 探索型可以插入1-2个人物故事
- 故事型可以增加1-2个历史背景段落

⚠️ **不建议混合**的场景：

- 不要同时使用3个以上的核心模块（会超时）
- 不要在探索型中使用过多的历史内容（违背探索主题）
- 不要在故事型中插入过多的景点介绍（偏离人物主线）

### 混合示例

**案例：历史 + 人物混合**

```
三、核心内容（10-12分钟）
├── 3.1 历史背景（深度）- 来自选项D (4分钟)
├── 3.2 人物故事一 - 来自选项C (3分钟)
├── 3.3 人物故事二 - 来自选项C (3分钟)
└── 3.4 历史遗迹详解 - 来自选项D (2分钟)
```

## 常见问题 FAQ

### Q1: 我不确定用哪个模块怎么办？

**A**: 使用选项A（标准模板）。它是最通用的，适用于80%的场景。

### Q2: 可以中途切换模块吗？

**A**: 可以，但要注意总时长。如果发现选错了，调整后续内容即可。

### Q3: 模块的时长可以调整吗？

**A**: 可以，所有时长都是建议值。根据实际内容灵活调整。

### Q4: 归档的旧模板还能用吗？

**A**: 可以。它们保存在 `Templates/Archive/` 目录，功能完全正常。

### Q5: 如何给建议反馈？

**A**: 在本指南末尾添加你的使用经验，帮助后续改进。
