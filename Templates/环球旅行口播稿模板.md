## 制作笔记

### 拍摄清单

- [ ] 开场震撼镜头（城市全景）
- [ ] 抵达过程（交通/入境）
- [ ] 街头氛围（建筑/人群）
- [ ] 历史遗迹/博物馆
- [ ] 当地人生活场景
- [ ] 人物采访/互动
- [ ] 小众景点

- [ ] 美食特写
- [ ] 个人出镜（反应/思考）
- [ ] 城市夜景/标志性镜头
- [ ] 空镜素材（后期用）

### 后期制作要点

1. **节奏控制**: 前3分钟必须有震撼镜头
2. **音乐选择**: 开场激昂 → 中段舒缓 → 升华部分感人
3. **字幕**: 关键信息、人名、地名必须加字幕
4. **转场**: 不同部分之间用地图或动画转场

### 常见问题

**Q: 时长不够怎么办？**
A: 扩展"文化特色"或"人物故事"部分，增加深度

**Q: 时长超了怎么办？**
A: 压缩"抵达与初印象"或"实用信息"部分

**Q: 没找到当地人采访怎么办？**
A: 用你的观察和体验代替，或拍摄街头人物纪实

## 模板使用指南

### 填写步骤

1. **先做调研**: 收集城市历史、文化、人物信息
2. **填写大纲**: 先把所有[ ]替换成具体内容
3. **撰写口播**: 逐段写出口播词，注意口语化
4. **标注画面**: 在每个段落标注需要的画面
5. **预估时间**: 每段标注预估时长，总计应在20-25分钟
6. **调整优化**: 根据实际情况调整各部分比重

### 写作技巧

- **开场**: 必须有反差或悬念，不要平铺直叙
- **故事**: 用具体人物的故事代替抽象描述
- **细节**: 用具体的观察细节代替笼统描述
- **情感**: 在个人思考部分真诚表达，不要煽情
- **实用**: 实用信息必须准确、有用、可操作
