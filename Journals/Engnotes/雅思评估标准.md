
在雅思口语（IELTS Speaking）评分里，**FC / LR / GRA / P** 是四个评分维度的缩写：

## 1) FC = Fluency & Coherence（流利度与连贯性）
看你说话是否**顺畅**、表达是否**有逻辑**、信息组织是否清楚。
- 流利：少长停顿、少频繁自我纠正、语速自然
- 连贯：回答有结构，能用连接词把观点串起来（because / so / however / for example 等）

## 2) LR = Lexical Resource（词汇资源）
看你词汇的**丰富度、准确度、搭配能力**，以及你会不会用同义替换来避免重复。
- 词汇范围：能否覆盖不同话题
- 搭配：make a decision / take responsibility 这类自然组合
- 准确：词用得对、不生硬
- 处理生词：不会的词能否换说法（paraphrase）

## 3) GRA = Grammatical Range & Accuracy（语法范围与准确性）
看你能否使用**多样句型**并保持**错误率可控**。
- 范围：简单句、并列句、从句、分词结构等是否都有
- 准确：时态、主谓一致、冠词、单复数等基础错误多不多
- 控制力：复杂句是否用得稳，不是“硬上导致全错”

## 4) P = Pronunciation（发音）
看你的发音是否**清晰可懂**、是否有自然的**语调/重音/节奏**，而不是“像母语者”。
- 清晰度：考官听不听得懂
- 重音与节奏：关键词重读、意群停顿
- 语调：陈述/疑问/强调的升降
- 连读弱读：更自然，但不是必须完美

## 写作 Writing（官方评分维度）
### Task 1：**TA / CC / LR / GRA**
1) **TA = Task Achievement（任务完成度）**  
- 是否覆盖所有要点（概述/主要特征/对比）  
- 数据是否选得对、是否有概括（overview）  
- 细节是否准确不过度解读

2) **CC = Coherence & Cohesion（连贯与衔接）**  
- 段落是否清晰（分段、主旨句）  
- 信息组织是否顺（对比/分类/时间线）  
- 连接词是否自然（不堆 moreover / furthermore）

3) **LR = Lexical Resource（词汇资源）**  
- 词汇范围与准确性（尤其学术常用词）  
- 搭配与同义替换（increase/rise/grow；significant/sharp/steady）  
- 拼写与用词是否自然

4) **GRA = Grammatical Range & Accuracy（语法范围与准确性）**  
- 句型多样性（从句/分词/被动/比较）  
- 基础错误率（时态、冠词、单复数、标点、句子完整性）

### Task 2：**TR / CC / LR / GRA**
- **TR = Task Response（回应任务）**：立场明确、论证充分、例子具体、没有跑题。

## 听力 Listening（客观题，但可用“能力维度”拆解）
> 听力最终按**正确题数**给分；下面是最常见的“丢分点维度化”。

1) **PF = Prediction & Focus（预测与注意力管理）**  
- 读题预测词性/范围（数字、名词、地点、时间）  
- 把注意力放在“即将出现的信息”而不是上一题

2) **PR = Paraphrase Recognition（同义替换识别）**  
- 听到的表达 ≠ 题目上的词（例如 “reduce cost” 对应 “cut expenses”）

3) **DH = Distractor Handling（干扰项处理）**  
- 典型：先给一个答案→立刻改口（actually / but / instead）

4) **DA = Detail Accuracy（细节精确）**  
- 单复数、冠词、单位、大小写、年份、电话号码、地址等

5) **FS = Form & Spelling（格式与拼写）**  
- 拼写错误=不得分；答案字数限制、是否需要复数等

## 阅读 Reading（客观题，但同样可拆解成能力维度）
1) **LS = Locating Skill（定位能力：scan）**  
- 用关键词/专有名词/数字快速定位，而不是通篇精读

2) **PM = Paraphrase Mapping（同义改写匹配）**  
- 题干用词与原文用词不同，但意思一致

3) **I = Inference（推断能力）**  
- Not Given / Yes-No-Not Given / 判断作者态度类题型核心

4) **QTS = Question-type Strategy（题型策略）**  
- 不同题型不同做法：匹配题、段落标题、T/F/NG、填空等

5) **TM = Time Management（时间管理）**  
- 不在难题上死磕；按段/按题型控时，保证三篇都做完

6) **VT = Vocabulary Tolerance（生词容忍度）**  
- 不要求懂每个词；抓句子逻辑关系（转折、因果、让步）更重要
