>我的观众目标是对关注于个人成长的重度用户. 高学识,高自驱力.所以请设计具体的提示词. 英文.1-3句话

## 优化提示词设计

```
You are designing a presentation for highly self-directed, intellectually engaged individuals 
focused on personal growth. Prioritize depth over breadth: emphasize non-obvious insights, 
systems thinking, and actionable frameworks that challenge conventional wisdom. 
Use data-driven evidence and integrate psychological/neurobiological foundations where relevant. 
Minimize motivational platitudes; instead, structure each slide around a specific insight, 
trade-off, or mental model that compounds value.
```

**或更简洁版本**：

```
Create a deck for ambitious, self-learning individuals. Prioritize counterintuitive insights, 
systems thinking, and frameworks over inspiration. Each slide should deliver depth—evidence-based reasoning, 
tradeoffs, or mental models—not motivation.
```

---

## 关键设计要素说明

|维度|设计逻辑|
|---|---|
|**避免的**|Motivational quotes, generic tips, surface-level positivity|
|**强调的**|Mental models, tradeoffs, evidence, system dynamics|
|**结构**|Insight → Evidence → Application, not problem → inspiration|
|**语言**|Precise, technical where appropriate, assume baseline competence|

这会让生成内容从"激励鸡汤"转向"可复用的认知工具"。
