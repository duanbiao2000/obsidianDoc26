"Steel-manned" 是一个固定搭配，意为“以最强、最合理的版本来陈述对方观点”，即对对方的论点进行最有力、最严谨的重构，而非简单地反驳其薄弱之处。
在这个句子中，“Steel-manned core thesis”指的是一种经过强化后的核心论点，即不是攻击原论点中最容易被驳倒的部分，而是将其提升到最具说服力和逻辑严密性的状态。
因此，这句话的意思是：该论点在最强状态下主张——在许多蚁种中，群体（蚁群）才是生物组织与适应性的主要单位，而非个体蚂蚁。
这种表述方式体现了对原理论的尊重与深化，而非片面批判。
