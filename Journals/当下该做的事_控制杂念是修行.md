**2026年3月3日 晴  法兰克福**

今天清晨，对着镜子洗脸时，又一次想起那句话：
“人若由思想之杂念缠绕，无所作为，就会被烦恼淹没。但只要控制这烦恼，人性中本来就有为他人着想的美好的心意就会出现，不仅让自己而且使周围的世界好起来。”

这句话像一面镜子，照出了我昨天和前天的状态。

昨天工作到深夜，遇到一个棘手的项目反馈——客户反复修改要求，团队有些疲惫。我当时心里生出抱怨：“为什么总变来变去？他们不懂我们的辛苦。”
那一刻，杂念就开始缠绕了。
怨恨、自我辩护、疲惫感像潮水一样涌上来。结果呢？
我效率下降，回复邮件时语气不自觉带了点生硬，晚上回家还辗转反侧，睡得不好。
完全被烦恼淹没，什么正向的事都没做成。
这就是“思想之杂念缠绕 → 无所作为 → 被烦恼淹没”的活生生例子。

今天早上反省时，我问自己：
“昨天的那些负面情绪，到底从哪里来？”
答案很清楚：从“我”的执着。从“我太辛苦了”“别人不理解我”“为什么是我倒霉”的小小自我中心。
这些杂念一多，心就浑浊了。镜子脏了，看什么都模糊。

稻盛先生教导我们：烦恼不是敌人，而是提醒。
它在告诉我——你的心又偏离“利他”了。
所以今天我决定：不再和这些感性的烦恼纠缠。
一旦觉察到抱怨、嫉妒、怨恨升起，就立刻停下来，深呼吸，然后把注意力拉回到“当下该做的事”和“能为别人带来什么价值”上。

今天具体做了这些事：

- 上午开会前，在心里默念：“今天的会议，不是为了证明自己正确，而是为了让团队和客户都更接近最好的结果。”
- 下午写报告时，刻意问自己：“这段话写出来，能否让读到的人少走弯路、少受挫折？”
- 晚上给一位远方的朋友回了消息，不是敷衍，而是真心写下：“最近看你有些累，有什么我能帮的吗？”

结果很奇妙。
当我把焦点从“我”转向“他人”时，那些杂念像云一样散了。
取而代之的是一种清爽、轻快的状态。
工作推进得比昨天顺畅许多，团队反馈也更积极。
甚至那位客户下午主动说：“今天沟通顺了很多，谢谢你的耐心。”

这让我再次确认：
人性深处，本来就有为他人着想的美好心意。
它不需要我们费力去“制造”，只需要我们把遮蔽它的东西——贪嗔痴、自我中心、杂念烦恼——主动控制住，它就会自然显现。
就像拨开乌云，太阳本来就在那里。

今天最深的感悟是：
控制烦恼，不是压抑情绪，而是把心转向更高的方向。
不是“我不能生气”，而是“我选择感恩”“我选择利他”“我选择全力以赴”。
一旦这么做，不仅自己内心平静，周围的世界也跟着变好一点点。

明天继续：

- 早起反省：昨天有没有一丝傲慢或偷懒？
- 全天努力：不亚于任何人的努力，但动机必须至善、无私。
- 晚上感恩：感谢今天能觉察、能控制、能让一点点善意流露。

活着，真是一件值得深深感谢的事。
因为只要心性向上提升一点，世界就会跟着明亮一点。

晚安。
明天也要以更好的心，面对新的一天。

—— Danny

### 3. 真实世界例子：六项精进的直接应用

稻盛晚年总结的**六项精进**，几乎就是这段话的实践版：

1. 付出不亚于任何人的努力 → 忙到没空烦恼
2. 要谦虚，不要骄傲 → 减少傲慢杂念
3. 要每天反省 → 主动觉察并控制负面情绪
4. 活着，就要感谢 → 感恩取代抱怨
5. 积善行、思利他 → 主动让“为他人着想”的心意显现
6. 不要有感性的烦恼 → 理性克制情绪化纠缠

他在日航重建时就是这样：不抱怨破产、不怨天尤人，而是每天反省、感恩员工、为员工福祉着想（利他），结果一年扭亏为盈，创历史最高利润。

### 4. 比较与对照：不同心性状态的结局

|心性状态|内在表现|行动结果|周围影响|长期命运|
|---|---|---|---|---|
|被杂念烦恼主导|利己、怨恨、比较、焦虑|拖延、分心、半途而废|冲突、孤立、被人疏远|恶性循环，越努力越挫败|
|主动控制烦恼后|利他、感恩、专注、谦虚|全力以赴、精益求精|信任、合作、感染他人|正向循环，命运自然好转|

**为什么稻盛强调“控制烦恼”而不是“消灭烦恼”？** 因为烦恼是人性的一部分（凡夫俗子难免），但**可以被驾驭**。消灭是妄想，控制是修行。
