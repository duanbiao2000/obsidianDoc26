## 核心社交 hack 术语中英对照

| 英文术语                    | 中文表达   | 简短中文解释                          |
| ----------------------- | ------ | ------------------------------- |
| Compliment Combo        | 赞美组合技巧 | 用一条具体真诚的赞美打开话题、降低陌生感。​        |
| Name Drop Resurrection  | 名字复活技巧 | 承认自己忘名或记错，通过幽默把对方名字“重新请出来”。​  |
| Mirror Move             | 镜像动作   | 轻微模仿对方的语气和能量，来建立同频和熟悉感。​      |
| 3-Second Rule of Entry  | 三秒开口规则 | 在三秒内行动或开口，避免过度犹豫导致错失社交机会。​    |
| Echo Hack               | 回声技巧   | 用自己的话重复对方关键点，让对方感到被认真倾听。​     |
| Vulnerability Vibe Bomb | 脆弱氛围炸弹 | 适度暴露自己的小脆弱，拉近心理距离、营造真诚氛围。​    |
| Exit Portal             | 退出门户   | 设计一个自然的结束语，让对话优雅收尾而不是突然中断。​   |
| Call-back Connection    | 回调式连接  | 后面聊天时再提到对方之前说过的点，加深“被记住”的感觉。​ |
| Curiosity Catalyst      | 好奇催化剂  | 抓住对方话里的小线索，靠提问把对话往更深处推进。​     |
| Social Battery          | 社交电量   | 把社交精力比喻成电量，意识到自己能量有限需要充电。​    |

## 常用形象短语中英对照

|英文短语|中文表达|简短中文解释|
|---|---|---|
|Social lockpick|社交开锁器|能解锁尴尬局面的一个动作或一句话。​|
|Armor of awkwardness|尴尬盔甲|因紧张而把自己包起来，难以自然交流的状态。​|
|Mental “join now” button|大脑里的“现在加入”按钮|逼自己立刻参与，而不是继续旁观和犹豫。​|
|Background extra|背景路人|只在场不参与，存在感很低的角色。​|
|On someone’s wavelength|在同一频率上|说话节奏、话题兴趣都很合得来。​|
|Polite prison of small talk|客气寒暄的监狱|一直停在无聊的表面话题，出不去也不深入。​|
|Conversational fire|对话之火|一段聊得很热烈、有延续性的谈话。​|
|Invisible thread|无形线索|能把一次对话拉长、拉深的共同点或细节。​|
|Graceful fade|优雅淡出|慢慢把对话收尾，而不是突然消失。​|
|Intentional curiosity|有意的好奇|主动带着好奇问问题，而不是机械地聊天。​|
