# Session Title Recommender

为 Claude Code 会话生成清晰、易辨识的标题，解决会话列表难以区分的问题。

## 使用场景

当你：

- 开始新的 Claude Code 对话前
- 想要更好地组织历史会话
- 需要从多个角度处理复杂任务

## 工作方式

描述你的任务意图，我会：

1. **生成主标题** - 基于你的描述生成一个推荐标题
2. **分析并行可能性** - 判断任务是否可以拆分为多个并行会话
3. **提供标题清单** - 如果可以并行，提供多个标题供选择

## 标题格式建议

| 前缀 | 用途 | 示例 |
|------|------|------|
| 【开发】 | 新功能开发、代码编写 | 【开发】创建用户认证模块 |
| 【Bug修复】 | 调试和修复问题 | 【Bug修复】修复登录超时问题 |
| 【配置】 | 工具配置、环境设置 | 【配置】设置 pre-commit hook |
| 【学习】 | 学习新技术、研究 | 【学习】分析 Claude 系统提示词 |
| 【优化】 | 性能优化、代码重构 | 【优化】重构数据库查询逻辑 |
| 【文档】 | 文档编写、整理 | 【文档】编写 API 使用指南 |
| 【调研】 | 功能调研、方案对比 | 【调研】对比 ORM 框架选型 |
| 【仓库维护】 | 仓库整理、格式化 | 【仓库维护】配置 markdownlint |

## 示例

**输入：**

```
我想配置 markdownlint 并修复所有格式问题
```

**输出：**

```
推荐标题：【配置】配置 markdownlint 并修复仓库格式问题

此任务可以拆分为多个并行会话：
1. 【配置】创建 .markdownlint.json 配置文件
2. 【配置】设置 pre-commit hook 自动运行 markdownlint
3. 【仓库维护】批量修复所有 markdown 文件格式
```
