# 字段速查表

> 快速查找正确的字段名。

## ⚡ 核心区别

| 特征 | 内置字段 (file.*) | 自定义字段 (frontmatter) |
|------|------------------|--------------------------|
| **存在性** | ✅ 所有文件都有 | ⚠️ 只有添加了的才有 |
| **null 检查** | ❌ 不需要 | ✅ **必须检查** |
| **示例** | `file.ctime`, `file.path` | `title`, `author`, `created` |

## 📅 时间字段对比

| 场景 | 使用字段 | 示例 |
|------|----------|------|
| **文件创建时间** | `file.ctime` | `WHERE file.ctime >= date(today)` |
| **文件修改时间** | `file.mtime` | `SORT file.mtime DESC` |
| **内容创建时间** | `created` | `WHERE created != null AND created >= ...` |
| **完成时间** | `completed` | `WHERE completed != null` |

## 🎯 正确 vs 错误示例

### 查询最近创建的文件

```dataview
// ❌ 错误：created 是自定义字段，不是所有文件都有
TABLE title, created
FROM ""
WHERE created >= date(today) - dur(7 days)

// ✅ 正确：使用 file.ctime（所有文件都有）
TABLE title, file.ctime
FROM ""
WHERE file.ctime >= date(today) - dur(7 days)
SORT file.ctime DESC
```

### 查询自定义元数据

```dataview
// ✅ 正确：自定义字段必须检查 null
TABLE title, author, rating
FROM "Books"
WHERE author != null AND rating != null
WHERE rating >= 4
SORT rating DESC
```

## 📋 常用内置字段

```dataview
file.ctime      // 创建时间（所有文件）
file.mtime      // 修改时间（所有文件）
file.path       // 完整路径（所有文件）
file.folder     // 文件夹（所有文件）
file.name       // 文件名（所有文件）
file.tags       // 所有标签（所有文件，可能为空）
file.size       // 文件大小（所有文件）
file.link       // 文件链接（所有文件）
```

## 🏷️ 常用自定义字段（模板中定义）

```dataview
title          // 书名/文章标题（Books, Journals）
author         // 作者（Books）
rating         // 评分（Books）
status         // 状态（Work, Projects）
priority       // 优先级（Work）
created        // 内容创建时间（所有模板）
completed      // 完成时间（Work, Books）
due            // 截止日期（Work）
```

## 💡 快速规则

### 规则 1：时间相关查询

**优先使用 `file.ctime`**，除非：

- 你需要记录"内容创建时间"而非"文件创建时间"
- 文件复制/移动后需要保持原时间

### 规则 2：null 检查

**内置字段**（`file.*`）：不需要

```dataview
WHERE file.ctime >= date(today)  // ✅ 正确
```

**自定义字段**：需要

```dataview
WHERE created != null AND created >= date(today)  // ✅ 正确
```

### 规则 3：混合使用

```dataview
// 同时使用内置字段和自定义字段
TABLE
  title,
  file.ctime AS "文件创建",
  created AS "内容创建",
  author
FROM "Books"
WHERE author != null  // 只需要检查自定义字段
SORT file.ctime DESC
```
