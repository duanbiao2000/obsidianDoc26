# 项目追踪仪表板

> 监控所有项目的状态、预算、里程碑和时间线。刷新以更新数据。

## 🚀 活跃项目

### 所有活跃项目

```dataview
TABLE
  phase AS "阶段",
  status AS "状态",
  start_date AS "开始日期",
  target_end AS "目标结束"
FROM "Projects"
WHERE status = "active"
SORT target_end ASC
```

### 优先级项目

```dataview
TABLE
  phase AS "阶段",
  start_date AS "开始日期",
  target_end AS "目标结束",
  budget AS "预算(美元)"
FROM "Projects"
WHERE status = "active"
WHERE priority = "P0" OR priority = "P1"
SORT target_end ASC
```

## 📅 里程碑追踪

### 即将到来的里程碑

```dataview
TABLE
  target_end AS "目标日期",
  phase AS "当前阶段"
FROM "Projects"
WHERE status = "active"
WHERE target_end != null AND target_end >= date(today) AND target_end <= date(today) + dur(30 days)
SORT target_end ASC
```

### 延期项目

```dataview
TABLE
  target_end AS "目标结束",
  actual_end AS "实际结束",
  durformat(actual_end - target_end, "d天") AS "延期"
FROM "Projects"
WHERE status = "completed"
WHERE actual_end != null AND target_end != null
WHERE actual_end > target_end
WHERE actual_end != null AND target_end != null AND actual_end > target_end
SORT (actual_end - target_end) DESC
```

## 💰 预算追踪

### 活跃项目预算

```dataview
TABLE
  budget AS "预算(美元)",
  (budget / 1000) AS "预算(K美元)"
FROM "Projects"
WHERE status = "active"
SORT budget DESC
```

### 预算偏差

```dataview
TABLE
  budget AS "预算(美元)",
  actual_cost AS "实际(美元)",
  (actual_cost - budget) AS "偏差(美元)",
  ((actual_cost / budget * 100) - 100) AS "偏差(%)"
FROM "Projects"
WHERE status = "completed"
WHERE actual_cost != null
SORT (actual_cost - budget) DESC
```

## 📈 项目统计

### 按状态分组

```dataview
TABLE
  count(rows) AS "项目数",
  sum(budget) AS "总预算(美元)",
  sum(actual_cost) AS "总实际(美元)"
FROM "Projects"
GROUP BY status
```

### 按阶段分组

```dataview
TABLE
  count(rows) AS "项目数"
FROM "Projects"
WHERE status = "active"
GROUP BY phase
```

### 按类别分组

```dataview
TABLE
  count(rows) AS "项目数"
FROM "Projects"
GROUP BY category
SORT count(rows) DESC
```

## 👥 项目团队

### 团队成员工作量

```dataview
TABLE
  count(rows) AS "参与项目数"
FROM "Projects"
FLATTEN team AS member
WHERE status = "active"
GROUP BY member.name
SORT count(rows) DESC
```

## ✅ 已完成项目

```dataview
TABLE
  completed AS "完成日期",
  category AS "类别",
  budget AS "预算(美元)",
  actual_cost AS "实际(美元)"
FROM "Projects"
WHERE status = "completed"
SORT completed DESC
LIMIT 10
```
