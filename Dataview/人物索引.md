# 人物索引

> 按国籍、职业、领域和时代索引所有人物笔记。刷新以更新数据。

## 🌍 按国籍索引

### 所有国籍

```dataview
TABLE
  count(rows) AS "人数"
FROM "People"
GROUP BY nationality
SORT count(rows) DESC
```

### 各国人物列表

```dataview
TABLE
  link(file.link, title) AS "人物",
  occupation AS "职业",
  born AS "出生年份",
  died AS "逝世年份"
FROM "People"
SORT nationality, born ASC
```

### 学术界

```dataview
TABLE
  link(file.link, title) AS "学者",
  nationality AS "国籍",
  born AS "出生年份",
  tags AS "标签"
FROM "People"
WHERE occupation INCLUDES ("学者" OR "教授" OR "科学家" OR "研究员")
SORT born ASC
```

## 🔗 关联人物

### 有互相关注的人物

```dataview
TABLE
  related_people AS "相关人物"
FROM "People"
WHERE related_people != null
FLATTEN related_people AS person
```

## 📊 统计信息

```dataview
TABLE WITHOUT ID
  count(rows) AS "总人数"
FROM "People"
GROUP BY true
```

### 按世纪统计

```dataview
TABLE
  (floor(born / 100) * 100) AS "世纪",
  count(rows) AS "人数"
FROM "People"
WHERE born != null
GROUP BY (floor(born / 100) * 100)
SORT (floor(born / 100) * 100) ASC
```
