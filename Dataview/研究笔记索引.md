# 研究笔记索引

> 按类型、领域和分类索引所有研究笔记。刷新以更新数据。

## 🔬 按研究类型

### 物种研究

```dataview
TABLE
  link(file.link, title) AS "物种",
  classification AS "分类",
  research_area AS "研究领域",
  status AS "状态"
FROM ""
WHERE type = "biology-research" AND subtype = "species"
SORT classification
```

### 实验研究

```dataview
TABLE
  link(file.link, title) AS "实验",
  created AS "创建日期",
  status AS "状态"
FROM ""
WHERE type = "biology-research" AND subtype = "experiment"
SORT created DESC
```

### 概念研究

```dataview
TABLE
  link(file.link, title) AS "概念",
```

## 📊 研究统计

### 按状态统计

```dataview
TABLE
  count(rows) AS "笔记数"
FROM ""
WHERE type = "biology-research"
GROUP BY status
```

### 按子类型统计

```dataview
TABLE
  count(rows) AS "笔记数"
FROM ""
WHERE type = "biology-research"
GROUP BY subtype
```

### 按研究领域统计

```dataview
TABLE
  count(rows) AS "笔记数"
FROM ""
WHERE type = "biology-research"
FLATTEN research_area AS area
GROUP BY area
SORT count(rows) DESC
```

## 🔗 关联笔记

### 有实验日志的研究

```dataview
TABLE
  file.lists AS "实验日志"
FROM ""
WHERE type = "biology-research"
WHERE file.lists != null
FLATTEN file.lists AS log
WHERE log.header INCLUDES "实验日志"
```

### 有后续实验的研究

```dataview
TABLE
  后续实验 AS "计划"
FROM ""
WHERE type = "biology-research"
WHERE 后续实验 != null
```
