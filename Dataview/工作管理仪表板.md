# 工作管理仪表板

> 追踪所有工作项目的状态、优先级和进度。刷新以更新数据。

## 🎯 当前工作

### 待办工作 (按优先级)

```dataview
TABLE
  priority AS "优先级",
  status AS "状态",
  due AS "截止日期",
  assignee AS "负责人"
FROM "Work"
WHERE status != "completed" AND status != "cancelled"
SORT priority ASC, due ASC
```

### 今日任务

```dataview
TABLE
  due AS "截止日期",
  priority AS "优先级"
FROM "Work"
WHERE status != "completed" AND status != "cancelled"
WHERE due <= date(today)
SORT priority ASC
```

### 本周到期

```dataview
TABLE
  priority AS "优先级",
  due AS "截止日期"
FROM "Work"
WHERE status != "completed" AND status != "cancelled"
WHERE due != null AND due > date(today) AND due <= date(today) + dur(7 days)
SORT due ASC, priority ASC
```

## 🚧 阻塞与问题

### 被阻塞的工作

```dataview
TABLE
  due AS "截止日期",
  assignee AS "负责人"
FROM "Work"
WHERE status = "blocked"
SORT due ASC
```

### 高优先级未完成

```dataview
TABLE
  priority AS "优先级",
  due AS "截止日期",
  estimated_hours AS "估算(小时)",
  actual_hours AS "实际(小时)"
FROM "Work"
WHERE (priority = "P0" OR priority = "P1")
WHERE status != "completed" AND status != "cancelled"
SORT priority ASC, due ASC
```

## 📊 工作统计

### 按状态分组

```dataview
TABLE
  count(rows) AS "数量",
  sum(estimated_hours) AS "总估算(小时)",
  sum(actual_hours) AS "总实际(小时)"
FROM "Work"
GROUP BY status
```

### 按优先级分组

```dataview
TABLE
  count(rows) AS "数量"
FROM "Work"
WHERE status != "completed" AND status != "cancelled"
GROUP BY priority
```

### 按负责人分组

```dataview
TABLE
  count(rows) AS "待办数量",
  sum(estimated_hours) AS "总估算(小时)"
FROM "Work"
WHERE status != "completed" AND status != "cancelled"
GROUP BY assignee
SORT count(rows) DESC
```

## ⏱️ 时间追踪

### 超时工作

```dataview
TABLE
  due AS "截止日期",
  (actual_hours - estimated_hours) AS "超时(小时)",
  (actual_hours / estimated_hours * 100 - 100) AS "偏差(%)"
FROM "Work"
WHERE status = "completed"
WHERE actual_hours > estimated_hours
SORT (actual_hours - estimated_hours) DESC
```

### 估算准确性

```dataview
TABLE WITHOUT ID
  (sum(actual_hours) / sum(estimated_hours) * 100 - 100) AS "平均偏差(%)"
FROM "Work"
WHERE status = "completed"
WHERE actual_hours != null AND estimated_hours != null
```

## 📅 最近完成

```dataview
TABLE
  completed AS "完成日期",
  priority AS "优先级",
  actual_hours AS "实际(小时)"
FROM "Work"
WHERE status = "completed"
SORT completed DESC
LIMIT 10
```
