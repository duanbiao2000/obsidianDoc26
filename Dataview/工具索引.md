# 工具索引

> 按类别、平台和状态索引所有工具笔记。刷新以更新数据。

## 🗂️ 按类别索引

### 所有类别

```dataview
TABLE
  count(rows) AS "工具数"
FROM "Tools"
GROUP BY category
SORT count(rows) DESC
```

### 命令行工具 (CLI)

```dataview
TABLE
  version AS "版本",
  platform AS "平台",
  status AS "状态",
  tags AS "标签"
FROM "Tools"
WHERE category = "cli"
SORT name ASC
```

### 图形界面工具 (GUI)

```dataview
TABLE
  version AS "版本",
  platform AS "平台",
  status AS "状态"
FROM "Tools"
WHERE category = "gui"
```

### 开发库 (Library/Framework)

```dataview
TABLE
  version AS "版本",
```

### 网络服务 (SaaS)

## 🔗 工具关系

### 有替代方案的工具

```dataview
TABLE
  related_tools AS "相关工具"
FROM "Tools"
WHERE related_tools != null
```

### 工具对比

```dataview
TABLE
  link(file.link, name) AS "工具",
  category AS "类别",
  status AS "状态"
FROM "Tools"
WHERE related_tools != null
FLATTEN related_tools AS tool
WHERE tool INCLUDES "vs" OR tool INCLUDES "alternative"
```
