# 总览仪表板

> 本页面提供所有笔记类型的全局视图。刷新以更新数据。

## 📊 笔记统计

```dataview
TABLE WITHOUT ID
  (rows.type) AS "类型",
  (rows.total) AS "总数",
  (rows.completed) AS "已完成",
  (rows.active) AS "进行中",
  (rows.pending) AS "待办"
FLATTEN rows
GROUP BY rows.type
```

```dataview
TABLE WITHOUT ID
  file.lists AS "待办事项总数"
FROM ""
WHERE file.lists.task
FLATTEN file.lists AS tasks
WHERE tasks.task
WHERE !tasks.completed
GROUP BY true
```

## 🎯 工作与项目概览

### 高优先级工作

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "工作",
  priority AS "优先级",
  status AS "状态",
  due AS "截止日期"
FROM "Work"
WHERE priority = "P0" OR priority = "P1"
WHERE status != "completed" AND status != "cancelled"
SORT due ASC
```

### 活跃项目

```dataview
TABLE WITHOUT ID
  link(file.link, name) AS "项目",
  phase AS "阶段",
  status AS "状态",
  target_end AS "目标日期"
FROM "Projects"
WHERE status = "active"
SORT target_end ASC
```

## 📚 阅读进度

### 正在阅读

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "书籍",
  author AS "作者",
  started AS "开始日期",
  tags AS "标签"
FROM "Books"
WHERE reading_status = "reading"
SORT started DESC
```

### 待阅读清单

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "书籍",
  author AS "作者",
  category AS "分类"
FROM "Books"
WHERE reading_status = "to-read"
SORT created DESC
LIMIT 10
```

### 已完成阅读 (Top 5 评分)

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "书籍",
  author AS "作者",
  rating AS "评分",
  completed AS "完成日期"
FROM "Books"
WHERE reading_status = "completed"
WHERE rating >= 4
SORT rating DESC, completed DESC
LIMIT 10
```

## 📓 日记统计

### 本月日记

```dataview
TABLE WITHOUT ID
  file.link AS "日期",
  length(file.lists) AS "笔记数"
FROM "Journals"
WHERE file.ctime >= date(today) - dur(30 days)
GROUP BY file.link
SORT file.ctime DESC
```

### 标签统计

```dataview
TABLE WITHOUT ID
  tags AS "标签",
  count(rows) AS "使用次数"
FROM "Journals"
FLATTEN file.tags AS tags
WHERE tags != null
GROUP BY tags
SORT count(rows) DESC
LIMIT 15
```

## 🔬 研究笔记

### 按研究领域

```dataview
TABLE WITHOUT ID
  research_area AS "领域",
  count(rows) AS "笔记数"
FROM ""
WHERE type = "biology-research" OR type = "research"
FLATTEN research_area AS research_area
WHERE research_area != null
GROUP BY research_area
SORT count(rows) DESC
```

## 使用说明

### 刷新数据
- **Obsidian Desktop**: 点击 `Dataview` 插件的刷新按钮，或使用快捷键
- **Live Preview**: 设置中开启"自动刷新"

### 自定义查询
1. 复制相关查询到其他笔记
2. 修改 `WHERE` 条件以过滤数据
3. 调整 `SORT` 和 `LIMIT` 以控制显示
