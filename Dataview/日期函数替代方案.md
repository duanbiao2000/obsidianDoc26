# 日期函数替代方案

> 当 `.relative()` 函数不工作时的替代方案。

## 🔴 问题

```dataview
// ❌ 报错
WHERE file.ctime >= date(today).relative(-7, "days")
```

**错误**: `Cannot call type 'null' as a function`

## ✅ 解决方案

### 方案 1: 使用 `dur()` 函数（推荐）

```dataview
TABLE title, file.ctime
FROM ""
WHERE file.ctime >= date(today) - dur(7 days)
SORT file.ctime DESC
```

### 方案 2: 使用固定日期

```dataview
TABLE title, file.ctime
FROM ""
WHERE file.ctime >= date("2026-03-01")  // 手动计算7天前
SORT file.ctime DESC
```

### 方案 3: 使用 `dateformat()` + 字符串比较

```dataview
TABLE title, file.ctime, dateformat(file.ctime, "yyyy-MM-dd")
FROM ""
WHERE dateformat(file.ctime, "yyyy-MM-dd") >= dateformat(date(today) - dur(7 days), "yyyy-MM-dd")
SORT file.ctime DESC
```

### 方案 4: 使用时间戳比较

```dataview
TABLE title, file.ctime
FROM ""
WHERE file.ctime >= date(now()) - dur(7 days)
SORT file.ctime DESC
```

### 方案 5: 分步骤查询

```dataview
// 先查看所有文件的 ctime
TABLE title, file.ctime
FROM ""
SORT file.ctime DESC
LIMIT 20
```

## 🎯 推荐方案

**使用 `dur()` 函数**：

// 最近7天创建的笔记

## 📋 常用时间范围模板

### 最近 N 天

```dataview
// 最近7天
WHERE file.ctime >= date(today) - dur(7 days)

// 最近30天
WHERE file.ctime >= date(today) - dur(30 days)

// 最近90天
WHERE file.ctime >= date(today) - dur(90 days)
```

### 本周

```dataview
// 从今天开始的未来7天
WHERE file.ctime >= date(today) AND file.ctime <= date(today) + dur(7 days)
```

### 本月

```dataview
// 方法1: 使用日期格式化
WHERE dateformat(file.ctime, "yyyy-MM") = dateformat(date(today), "yyyy-MM")

// 方法2: 使用时间范围
WHERE file.ctime >= dateformat(date(today), "yyyy-MM-01")
```

### 本年

```dataview
WHERE dateformat(file.ctime, "yyyy") = dateformat(date(today), "yyyy")
```

## 🚨 可能的原因

### 1. Dataview 版本问题

检查 Dataview 插件版本：

- 打开设置 → 社区插件 → Dataview
- 查看版本号
- 更新到最新版本

### 2. 损坏的文件

某些笔记可能有损坏的 frontmatter 或元数据。

**解决方案**:

```dataview
// 查找可能有问题的文件
TABLE title, file.ctime, typeof(file.ctime)
FROM ""
WHERE typeof(file.ctime) != "date"
```

### 3. 时区问题

日期计算可能受时区影响。

**解决方案**: 使用 `date(now())` 替代 `date(today)`

### 4. Obsidian 缓存问题

**解决方案**:

1. 关闭 Obsidian
2. 删除 `.obsidian/workspace` 或 `.obsidian/workspace.json`
3. 重新打开 Obsidian
4. 等待 Dataview 重建索引
