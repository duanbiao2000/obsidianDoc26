# 排除特定目录的查询

> 如何在 Dataview 查询中排除特定文件夹（如 agents-main）。

## 🎯 方法1: 使用 WHERE + file.folder（推荐）

```dataview
TASK
WHERE !completed
WHERE due <= date(today)
WHERE !contains(file.folder, "agents-main")
GROUP BY file.folder
```

### 说明

- `!contains(file.folder, "agents-main")` - 排除文件夹路径中包含 "agents-main" 的任务
- 适用于排除单个或多个文件夹

---

## 🎯 方法2: 使用 FROM + WHERE 组合

```dataview
TASK
FROM "Work" OR "Projects" OR "Journals"
WHERE !completed
WHERE due <= date(today)
GROUP BY file.folder
```

### 说明

- `FROM "Work" OR "Projects"` - 只查询指定的文件夹
- 适用于只包含特定目录

---

## 🎯 方法3: 排除多个文件夹

```dataview
TASK
WHERE !completed
WHERE due <= date(today)
WHERE !contains(file.folder, "agents-main")
WHERE !contains(file.folder, ".agent-state")
WHERE !contains(file.folder, ".claude")
GROUP BY file.folder
```

### 说明

- 添加多个 `WHERE !contains()` 条件
- 每个条件排除一个文件夹

---

## 🎯 方法4: 使用正则表达式（高级）

```dataview
TASK
WHERE !completed
WHERE due <= date(today)
WHERE file.folder !~ /^agents-main/
GROUP BY file.folder
```

### 说明

- `file.folder !~ /^agents-main/` - 使用正则表达式排除
- `!~` 表示"不匹配"
- 适用于复杂的匹配规则

---

## 🎯 方法5: 只查询一级目录（排除隐藏目录）

```dataview
TASK
WHERE !completed
WHERE due <= date(today)
WHERE !startswith(file.folder, ".")
GROUP BY file.folder
```

### 说明

- `!startswith(file.folder, ".")` - 排除以 "." 开头的文件夹
- 适用于排除所有隐藏文件夹（如 .agent-state, .claude, .git）

## 💡 最佳实践

### 1. 明确指定包含的目录（推荐）

```dataview
// ✅ 推荐：明确指定
TASK
FROM "Work" OR "Projects" OR "Journals"
WHERE !completed
WHERE due <= date(today)
```

**优点**:

- 更快（只查询需要的目录）
- 更安全（不会意外包含其他目录）
- 更清晰（一眼看出查询范围）

### 2. 排除系统/临时目录

```dataview
// ✅ 推荐：排除隐藏目录
TASK
WHERE !completed
WHERE due <= date(today)
WHERE !startswith(file.folder, ".")
GROUP BY file.folder
```

**优点**:

- 自动排除所有隐藏目录
- 不需要手动维护排除列表

## 🚨 常见错误

### 错误1: 在 FROM 中使用负向查询

```dataview
// ❌ 错误：Dataview 不支持 FROM NOT
TASK
FROM NOT "agents-main"
WHERE !completed
```

**正确做法**:

```dataview
// ✅ 正确：使用 WHERE 过滤
TASK
WHERE !completed
WHERE !contains(file.folder, "agents-main")
```

### 错误2: 排除路径不完整

```dataview
// ❌ 可能不工作：如果路径是 "Projects/agents-main"
TASK
WHERE file.folder != "agents-main"
```

**正确做法**:

```dataview
// ✅ 正确：使用 contains 匹配任意位置
TASK
WHERE !contains(file.folder, "agents-main")
```

## 📝 实用示例

### 今日任务（排除系统目录）

```dataview
TASK
WHERE !completed

WHERE !startswith(file.folder, ".")
GROUP BY file.folder
```

### 本周到期的任务（排除 agents）

```dataview
TASK
WHERE !completed
WHERE due <= date(today) + dur(7 days)
WHERE !contains(file.folder, "agents-main")
WHERE !contains(file.folder, ".agent-")
GROUP BY file.folder
```

### 高优先级任务（只查工作目录）

```dataview
TASK
FROM "Work" OR "Projects"
WHERE !completed
WHERE due <= date(today) + dur(7 days)
WHERE priority = "P0" OR priority = "P1"
GROUP BY file.folder
```
