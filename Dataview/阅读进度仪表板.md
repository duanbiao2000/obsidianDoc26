# 阅读进度仪表板

> 追踪阅读清单、阅读进度和书籍评分。刷新以更新数据。

## 📖 正在阅读

### 当前阅读列表

```dataview
TABLE
  author AS "作者",
  started AS "开始日期",
  tags AS "标签"
FROM "Books"
WHERE reading_status = "reading"
SORT started DESC
```

### 阅读进度（按天数）

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "书籍",
  author AS "作者",
  durformat(date(today) - started, "d天") AS "已读天数"
FROM "Books"
WHERE reading_status = "reading"
WHERE started != null
SORT started DESC
```

## 📚 待阅读清单

### 按优先级排序

```dataview
TABLE
  author AS "作者",
  category AS "分类",
  tags AS "标签"
FROM "Books"
WHERE reading_status = "to-read"
SORT created DESC
```

### 按推荐来源

```dataview
TABLE
  count(rows) AS "书籍数"
FROM "Books"
WHERE reading_status = "to-read"
GROUP BY recommended_by
SORT count(rows) DESC
```

### 按分类

```dataview
TABLE
  count(rows) AS "书籍数"
FROM "Books"
WHERE reading_status = "to-read"
GROUP BY category
SORT count(rows) DESC
```

## ✅ 已完成阅读

### 高分书籍 (4-5星)

```dataview
TABLE
  author AS "作者",
  rating AS "评分",
  completed AS "完成日期",
  category AS "分类"
FROM "Books"
WHERE reading_status = "completed"
WHERE rating >= 4
SORT rating DESC, completed DESC
```

### 按分类浏览

```dataview
TABLE
  link(file.link, title) AS "书籍",
  author AS "作者",
  rating AS "评分",
  completed AS "完成日期"
FROM "Books"
WHERE reading_status = "completed"
GROUP BY category
```

### 最近完成

```dataview
TABLE
  author AS "作者",
  rating AS "评分",
  completed AS "完成日期"
FROM "Books"
WHERE reading_status = "completed"
SORT completed DESC
LIMIT 10
```

## 📊 阅读统计

### 阅读状态概览

```dataview
TABLE
  count(rows) AS "书籍数"
FROM "Books"
GROUP BY reading_status
```

### 平均评分

```dataview
TABLE WITHOUT ID
  (sum(rating) / count(rows)) AS "平均评分"
FROM "Books"
WHERE reading_status = "completed"
WHERE rating != null
```

### 按分类统计

```dataview
TABLE
  count(rows) AS "书籍数",
  sum(rating) AS "总评分"
FROM "Books"
WHERE reading_status = "completed"
GROUP BY category
SORT count(rows) DESC
```

### 阅读速度（页/天）

```dataview
TABLE WITHOUT ID
  link(file.link, title) AS "书籍",
  author AS "作者",
  pages AS "页数",
  durformat(completed - started, "d天") AS "阅读天数",
  (pages / (completed - started)) AS "页/天"
FROM "Books"
WHERE reading_status = "completed"
WHERE completed != null AND started != null AND pages != null
WHERE (completed - started) > 0
SORT (pages / (completed - started)) DESC
```

## 🏆 年度阅读目标

```dataview
TABLE WITHOUT ID
  count(rows) AS "已完成",
  (50 - count(rows)) AS "剩余目标",
  (count(rows) / 50 * 100) AS "完成度(%)"
FROM "Books"
WHERE reading_status = "completed"
WHERE completed != null AND completed >= date("2026-01-01")
GROUP BY true
```
